== Description ==

Control what pages your widgets appear on. Based on Widget Visibility module, from <a href="http://wordpress.org/plugins/jetpack/">Jetpack plugin</a>. Created by <a href="http://www.labitacoradeltigre.com/">Eduardo Larequi</a>, mantained by <a href="http://boluda.com">Joan Boluda</a>.

== Installation ==

1. Upload 'widget-visibility-without-jetpack.php' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
